# Gowtham0992 Link

## How do I install these formulae?

`brew install gowtham0992/link/<formula>`

Or `brew tap gowtham0992/link` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "gowtham0992/link"
brew "<formula>"
```
